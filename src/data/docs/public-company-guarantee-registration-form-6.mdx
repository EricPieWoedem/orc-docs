
## Step 6: Attachment

![Attachment Section|2000x1238](/guar28.png)

This is the final step of the registration process, where the user can upload any additional relevant documents related to the registration.

### Key Features

- **Information Banner:**
  - Prompts the user to include any other relevant documents.
- **Upload Area:**
  - A drag-and-drop interface is provided for uploading files. The user can also click to open a file browser.
- **Navigation:**
  - **Previous Step:** Allows the user to return to the Beneficial Owner section.
  - **Submit:** Finalizes and submits the entire registration application.

After uploading any necessary documents, the user should click **Submit** to complete the application and proceed to the final review.
