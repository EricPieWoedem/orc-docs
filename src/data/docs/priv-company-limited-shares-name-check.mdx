# Name Availability Check
This step helps ensure your desired company name is available.

### ✅ Step 1: Type your desired entity name into the search box.

### ✅ Step 2: Click "Check Availability".

![Setp 1&2:|2000X1238](/priv-limited7.png)

### ➡️ If the name is available , you'll see: ***"The name is available and ready to use"***

![Setp 1&2:|2000X1238](/priv-limited8.png)

### ➡️ If not, it says: *"Sorry you can't proceed with this name because, The name is already in use"*

![Setp 1&2:|2000X1238](/priv-limited9.png)

### ✅ Step 3: If unavailable, type a new name and click on "Check Availability.

### ✅ Step 4: Once you find an available name, click "Continue" to move on with the application.
