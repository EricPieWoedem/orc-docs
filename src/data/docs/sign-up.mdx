# The Sign Up Page

This document details the components, functionality, and user interaction flows for the sign up page. This page serves as the gateway for new users to access the system.

![Sign Up Page|2000X1238](/sign-up-page.png)

## Title
**"Sign Up"**: A prominent heading at the top of the content area, designed to welcome new users to sign up.

## Core Sign-Up Options
The page offers two primary methods for user registration:

### Sign Up Using Tax Identification Number (TIN)
Clicking this button switches the registration flow to one specifically designed for TIN-based sign-up. This involves an input field for the TIN.
This option caters to non-Ghanaian users who are yet to register and will be identified using the Tax Identification Number they input into the system.

Sign Up Button:
Upon clicking, this button initiates the registration process using the entered Ghana Card Number.
