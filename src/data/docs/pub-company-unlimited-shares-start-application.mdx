# Registering a Public Company Unlimited by Shares

## Overview
A public company unlimited by shares is a body corporate formed without any restriction on the number of members per its constitution and its liabilities are unlimited.
## Start Application

###  Step 1: Starting from the Dashboard

After logging in to the user account:

- The user should hover over the "Registration and Incorporation" tab.

- The user should click on "Register a Business Entity".

![Start Application|2000X1238](/priv-limited.png)


###  Step 2: Choose Entity Type

The user will be directed to a page displaying the different types of entities such as:

*Business Name*

*Partnership*

*Private Company Limited by Shares (and others)*

**The user should select: Public Company Unlimited by Shares**

![Setp 2:|2000X1238](/priv-limited50.png)

###  Step 3: Entity Overview Page

The user will see a brief description and an informative video about the selected entity.

**The user should click the "Start Application" button to proceed.**

![Setp 3:|2000X1238](/priv-limited3.png)

###  Step 4: Process Summary Page

This page outlines the steps involved in completing the user registration.

**The user should tick the checkbox: "I have read and understood" and click the "Proceed" button.**
![Setp 4:|2000X1238](/priv-limited4.png)

###  Step 5: Basic Requirements Page

This page lists the documents and information required for registration.

**The user should tick the checkbox: "I have all relevant documents" and click the "Proceed" button.**
![Setp 5:|2000X1238](/priv-limited5.png)

###  Step 6: Choose Registration Service Type

The user should select between two options:

Standard – GHS 540.00

Prestige – GHS 1,540.00

**After choosing, the user should click the "Start Application" button.**
![Setp 6:|2000X1238](/priv-limited6.png)
